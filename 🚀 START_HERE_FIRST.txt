          HOW TO USE (Pick the easiest method for you)
================================================================================

METHOD 1: SUPER EASY (Recommended for beginners)
------------------------------------------------
Just run this command:

    python3 quick_start.py

Then choose:
- Option 1: Web interface (click and upload)
- Option 2: Analyze your Excel file
- Option 3: Run a test


METHOD 2: WEB INTERFACE (Best for multiple files)
--------------------------------------------------
Launch the web app:

    streamlit run app.py

Then:
1. Browser opens to http://localhost:8501
2. Upload your Excel file (drag & drop)
3. Click "Start Analysis"
4. Download beautiful formatted results


METHOD 3: COMMAND LINE (Fast & simple)
---------------------------------------
Analyze any Excel file:

    python3 main.py your_file.xlsx

The results will be saved automatically with a timestamp.


================================================================================
          WHAT YOU NEED (Excel file format)
================================================================================

Your Excel file should have 3 columns:

┌─────────────┬──────────────────────────────────────┬──────────┐
│ MPN         │ Model Description                    │ Quantity │
├─────────────┼──────────────────────────────────────┼──────────┤
│ ABC-12345   │ Industrial servo motor 5HP           │ 100      │
│ XYZ-67890   │ Precision ball bearing assembly      │ 250      │
│ DEF-11111   │ PLC controller with touchscreen      │ 50       │
└─────────────┴──────────────────────────────────────┴──────────┘

Column names are auto-detected - variations like "Part Number", 
"Description", "Qty" will work fine!

📁 See example_input.xlsx for a sample file


================================================================================
          WHAT YOU GET (Results)
================================================================================

Your output Excel file contains 2 sheets:

📊 SUMMARY SHEET:
   - Quick overview of all parts
   - Top manufacturer for each
   - Credibility scores (color-coded)

📋 DETAILED ANALYSIS SHEET:
   - All manufacturers found (3-5 per part)
   - Individual credibility scores
   - Strengths for each manufacturer
   - Considerations (MOQ, lead times, certifications)
   - AI-powered recommendations

🎨 COLOR CODING:
   🟢 Green (≥80)  = High credibility - Excellent choice!
   🟡 Yellow (60-79) = Medium credibility - Good option
   🔴 Red (<60)     = Low credibility - Verify first

📁 See test_results.xlsx for a sample output
